# 숫자 야구 게임
## 요구 사항
* [x] 플레이어 수와 컴퓨터 수가 일치하는지 확인 기능
* [x] 스트라이크, 볼을 결과 객체로 구성
* [x] 랜덤 숫자를 만드는 기능
* [ ] 중복된 숫자가 있는지 확인하는 기능
* [ ] 문자 숫자로 변환하는 기능
* [x] 플레이어 입력 기능
* [ ] 결과 출력 기능

## 객체
* 플레이어
  * 3개의 숫자 리스트
* 컴퓨터
  * 랜덤한 3개의 숫자 리스트
* 심판
  * 비교 메소드
* 결과
  * 스트라이크, 볼
* 입력
    * 숫자 문자열 3자리(1~9)
* 출력
    * 스트라이크, 볼, 낫싱 출력
